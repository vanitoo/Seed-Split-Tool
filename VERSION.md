# Version

Current release: **v0.4.0**

Release date: **2026-07-22**

## Release summary

Version 0.4.0 turns Seed Split Tool into a complete three-workflow browser application:

- generate a BIP-39 seed phrase;
- split a seed phrase or another secret;
- verify and recover the original secret.

The release includes SLIP-39, Banana Split, Generic SST1, Wallet fingerprint, contextual help, guided verification, smooth navigation, and GitHub Pages deployment.

## Stability

This is a pre-1.0 release intended for development, testing, and carefully controlled use.

Before version 1.0, the project will receive a broader architecture, dependency, browser compatibility, accessibility, UX, and security revision.
